# Sail XISA

This repository contains a formal specification of [XISA](https://xsightlabs.com/switches/xisa) (public [white paper](https://cdn.sanity.io/files/eqivwe42/production/affd0d0005566d4d8c50e05eff7fb60a43049a9f.pdf)), written in [Sail](https://github.com/rems-project/sail). This project is inspired by the [Sail RISC-V model](https://github.com/riscv/sail-riscv).
